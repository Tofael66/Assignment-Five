What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming API for HTML and XML documents.
It defines the logical structure of documents and the way a document is accessed and manipulated.

What is event delegation in the context of the DOM, and why is it useful?

In English Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.
It allows us to attach a single event listener for elements that exist now or in the future.
